# Minghui Yao's Homepage

This is a simple text-only GitHub Pages personal homepage.

## How to use

1. Create a repository named `547235344y.github.io` on GitHub.
2. Upload all files in this folder to the repository root.
3. Open `https://547235344y.github.io/` after GitHub Pages is enabled.

## Files

- `index.html`: main homepage
- `404.html`: simple not-found page
- `.nojekyll`: disables Jekyll processing
